## 简述
这是一个基于Python的LX Music后端实现，与LX Music相关的只有请求格式，其余全部重写  
项目基于 MIT 开源许可证开源于[Github](https://github.com/MeoProject/lx-music-api-server)上  
所调用API接口为网络上查找  
音乐平台不易，请尊重版权，支持正版。
